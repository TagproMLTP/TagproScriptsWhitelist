Guide

Here is a list of some different SmartMacros that you can use:

  • {?} - Splits the macros into different strings to be chosen at random. Example: "This might be said{?}Or this might be said!"
  • {mypups} - Returns the powerups you currently have. Example: "Coming in hot with {mypups}!"
  • {viewopp} - Returns the number of enemy players in view. Example: "{viewopp} in base!"
  • {s} - Returns the current number of seconds on the game clock. You can also use {s+n} to add seconds to this time. Example: "PUPs in 10 seconds! :{s+50}"
  • {ping} - Returns your current ping. Example: "Laaaaaggggg. {ping} ping!"
  • {myLocation} - Returns your current location on the map.

You can also create your own SmartMacro system using Toggles. Just add a toggle and include the toggle name in braces ({name}) in your macro to use the keys you select to change the string's value.
